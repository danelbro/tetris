# Tetris
A tetris clone written using C++, SDL, and my framework utilityKnife.

## Build instructions ##

### Linux ###

You need: 
* git
* CMake (3.27+)
* GCC

```
git clone --recursive https://github.com/danelbro/tetris.git
cd tetris
./config.sh [Debug/Release]
./build.sh [Debug/Release]
./run.sh [Debug/Release]
```
